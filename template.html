Grundlagen

  • Unter HTML5 wird kein Plugin wie der Flashplayer benötigt
  • Videos werden mithilfe des Video Befehls in die Website eingebunden
  • Videos können in 3 Formaten abgespielt werden
  • Nicht jeder Browser unterstützt jedes Format
  • Bis 2016 soll webm als Standard eingeführt werden

Der <video> Befehl

 
    <video "hier können Attribute eingetragen werden">
    <src src="Pfad der Datei" type="Dateitype der Datei"z.B.: type=video/mp4 / >
    <src src="Pfad der Datei" anderer Dateityp" z.B.: type=video/webm / >
    </video>
  • Videos können entweder im mp4 Format, im .ogg Format oder im webm Format abgespielt werden
  • Innerhalb des Video Befehls können 3 Quellen angegeben werden
  • CSS3 und Java Script Befehle können nun mit dem Video Tag benutzt werden

Attribute des Video Befehls

  • <width>: Verändert die Breite des Videos
  • <height>: Verändert die Höhe des Videos
  • <autoplay>: Das Video startet, sobald die Webseite aufgerufen wurde
  • <controls>: Der Nutzer kann das Video pausieren oder vor- und zurückspulen
  • <autobuffer>: Zwischenspeicherung wird aktiviert
  • <poster>: Ein Bild das angezeigt wird, wenn es noch nicht gestartet wurde
